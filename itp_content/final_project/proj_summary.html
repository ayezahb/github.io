Horizons: Project Summary

Topic

Håfa Adai! You are probably wondering, “What is Horizons?”, well, Horizons is a local mentorship organization for Guam high school students interested in receiving a post-secondary education at off-island institutions. The organization is made up of a group of local alumni from the Academy of Our Lady of Guam, St. John’s School, and Father Dueñas Memorial School. The organization aims to mentor high school students to gain a better understanding of applying to and attending institutions off-island, become strong applicants, and develop their confidence as applicants.

Purpose

Horizons is a fairly new organization and is still learning to reach out to a wider audience. This website was developed for the purpose of replacing the current website that Horizons uses (on Google Sites) and creating a more personalized (and developed) version of it. Furthermore, this website will serve as one of the main sources to receive Horizons’s updates and information. This website is meant for both applicants for a mentorship (mentee or mentor) who are looking to learn more about the organization.

Extras Used:

iframes

I used an iframe on the Learn More page of my website. I first used it to display the PDFs of Horizons’s past meetings, then I used it to embed our Instagram page.

HTML5 Media Tags

I also used media tags (for videos) on the website’s Learn More page to add a little bit of animation to the page. The four videos are located in #main_body and surround the text.

Transitions & Transformations

Homepage

The Horizons logo has a little floating animation, made possible by translating the image.

The “Work with Members from these Universities” effect, where the universities smoothly pop out from the main cover image (“Work with Members from these Universities”), was made possible through transform’s translate CSS value, and transition.

Services page

Upon clicking the images (bubbles), they smoothly transition and translate to one side from the middle of the page.

Board page

The same floating animation effect from the homepage is applied to the board member’s image - made possible through transform’s translate CSS value.

Members page

Here, the member’s image has a 3D image effect made possible through transform’s translate CSS value and transition.

JS or jQuery Plugin

In the homepage, I used a jQuery plugin called “slick” to create an image carousel for the college advice section.

JS or jQuery

I used jQuery in the website’s Services page. When you click on one of the images or “bubbles”, the jQuery event’s “click” function will be activated and apply CSS changes through making the images slide and making the words appear.
